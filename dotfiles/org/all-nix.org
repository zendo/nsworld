* Functions
** importModule
Return a list of full paths for all entries in the given directory, excluding default.nix
#+begin_src nix
importModule' =
  path:
  builtins.readDir path
  |> lib.attrNames
  |> lib.subtractLists [ "default.nix" ]
  |> map (n: path + "/${n}");
#+end_src
